The $5Website

This project was made with speed and ease in mind. Turning over a nice website for cheap. Customizing the website is a breeze but there are some things you need to provide.

First, we will need a title for your site. You can also throw in a short tagline if you want.

Then a logo. If you don't have a logo, we can just put a name on there. Toss in all the pictures you want to put in the site too. We can put as many as you want.

You can also provide all the links for your social media profiles so people can learn more about you.

Finally, pick out some colors for your theme:

  • The main color for your site.
  • An accent color.
  • Also an optional highlight color if you want.

If you have the content ready, we can slap that on too but you can easily do that if you want.

Feel free to get in touch if you need more info. Cheers!
